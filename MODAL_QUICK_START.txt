🎯 THREE WAYS MODALS OPEN/CLOSE
═════════════════════════════════════════════════════════════════════════════

METHOD 1: CLICK UPDATE BUTTON
─────────────────────────────
<button type="button" class="btn-danger" data-modal="changePasswordModal">
  Change Password
</button>

When clicked:
1. JavaScript detects data-modal attribute
2. Modal.openModal() called automatically
3. Modal fades in with animation
4. Background darkens
5. Scroll disabled


METHOD 2: CLICK X BUTTON TO CLOSE
──────────────────────────────────
<button type="button" class="modal-close">&times;</button>

When clicked:
1. JavaScript detects modal-close class
2. Modal.closeModal() called automatically
3. Modal fades out smoothly
4. Background scroll restored


METHOD 3: PRESS ESCAPE KEY
──────────────────────────
Press ESC key while modal is open

Effect:
1. All open modals close
2. Smooth fade-out animation
3. Background scroll restored

🎯 HOW TO USE
═════════════════════════════════════════════════════════════════════════════

USING IN YOUR HTML:

1. Create Modal HTML:
   <div id="myModal" class="modal">
     <div class="modal-content">
       <div class="modal-header">
         <h2>Title</h2>
         <button class="modal-close">&times;</button>
       </div>
       <div class="modal-body">
         <!-- Content -->
       </div>
       <div class="modal-footer">
         <button onclick="Modal.closeModal('myModal')">Cancel</button>
       </div>
     </div>
   </div>

2. Create Trigger Button:
   <button data-modal="myModal">Open Modal</button>

3. That's it! JavaScript handles everything.


PROGRAMMATIC USAGE:

Opening Modal:
Modal.openModal('changePasswordModal');

Closing Modal:
Modal.closeModal('changePasswordModal');

Close All Modals:
Modal.closeAllModals();
