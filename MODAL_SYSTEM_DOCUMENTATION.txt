🚀 MODAL JAVASCRIPT API
═════════════════════════════════════════════════════════════════════════════

The Modal object provides simple methods to manage modals:

1. Modal.openModal(modalId)
   ────────────────────────
   Purpose: Opens a modal by ID
   Usage: Modal.openModal('changePasswordModal')
   Effects:
   - Shows modal with fade-in animation
   - Prevents background scroll
   - Disables body overflow
   
2. Modal.closeModal(modalId)
   ─────────────────────────
   Purpose: Closes a specific modal
   Usage: Modal.closeModal('changePasswordModal')
   Effects:
   - Hides modal with fade-out
   - Re-enables background scroll
   - Clears any form data if needed

3. Modal.closeAllModals()
   ──────────────────────
   Purpose: Closes all open modals
   Usage: Modal.closeAllModals()
   Effects:
   - Closes every modal on page
   - Restores scroll functionality
   - Used for Escape key handler


🎯 MODAL TRIGGER METHODS
═════════════════════════════════════════════════════════════════════════════

METHOD 1: Data Attribute (Recommended)
──────────────────────────────────────
HTML:
<button type="button" class="btn-danger" data-modal="changePasswordModal">
  Change Password
</button>

JavaScript automatically detects and opens modal when clicked.
Advantage: No inline onclick handlers, cleaner HTML


METHOD 2: Direct JavaScript Call
───────────────────────────────────
JavaScript:
document.getElementById('myButton').addEventListener('click', function() {
  Modal.openModal('changePasswordModal');
});

Advantage: More control over button behavior


METHOD 3: Form Link
────────────────────
HTML:
<a href="#changePasswordModal" class="modal-trigger">Change Password</a>

Advantage: Semantic HTML, works without JS


🎨 MODAL STRUCTURE
═════════════════════════════════════════════════════════════════════════════

Basic Modal HTML Structure:

<div id="modalId" class="modal">
  <div class="modal-content">
    
    <!-- Header Section -->
    <div class="modal-header">
      <h2 class="modal-title">Modal Title</h2>
      <button type="button" class="modal-close">&times;</button>
    </div>
    
    <!-- Body Section -->
    <div class="modal-body">
      <!-- Form or content goes here -->
    </div>
    
    <!-- Footer Section -->
    <div class="modal-footer">
      <button type="button" class="btn btn-secondary">Cancel</button>
      <button type="submit" class="btn btn-primary">Submit</button>
    </div>
    
  </div>
</div>

Modal Content Structure:
- .modal: Full screen overlay container
- .modal-content: White card with shadow
- .modal-header: Title + close button
- .modal-body: Main content area
- .modal-footer: Action buttons

🔧 CLOSE MECHANISM DETAILS
═════════════════════════════════════════════════════════════════════════════

MULTIPLE CLOSE METHODS:

1. X Button Click
   Code: <button class="modal-close">&times;</button>
   Handler: Automatic (JavaScript)
   Effect: Closes modal, restores scroll

2. Backdrop Click
   Code: Click outside modal content
   Handler: Automatic (event.target === modal check)
   Effect: Closes modal when clicking overlay

3. Escape Key
   Code: Press ESC key
   Handler: Automatic (keydown listener)
   Effect: Closes all open modals

4. Cancel Button
   Code: <button onclick="Modal.closeModal('modalId')">Cancel</button>
   Handler: Manual call to Modal.closeModal()
   Effect: Closes specific modal

5. Form Submission
   Code: On submit event
   Handler: Manual in form submit handler
   Effect: Closes modal after processing
